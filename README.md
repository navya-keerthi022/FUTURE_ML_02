
# FUTURE_ML_02 - Support Ticket Classification & Prioritization

## 📌 Track Information

* Track Code: ML
* Task Number: 02

## 📊 Project Overview

This project focuses on building a Machine Learning system that automatically classifies customer support tickets and assigns priority levels. The system helps businesses streamline support operations and improve response time.

## 🎯 Objectives

* Classify tickets into categories (Technical, Billing, Account, etc.)
* Predict priority levels (High, Medium, Low)

## 🛠️ Technologies Used

* Python
* Pandas
* NLTK
* Scikit-learn
* Matplotlib

## ⚙️ Features

* Text preprocessing (cleaning, stopword removal)
* TF-IDF feature extraction
* Ticket classification using Logistic Regression
* Priority prediction system
* Model evaluation (Accuracy, Classification Report)
* Data visualization (graphs)

## 📈 Model Details

* Algorithm: Logistic Regression
* Feature Extraction: TF-IDF Vectorization

## 🚀 How to Run

1. Clone the repository
2. Open Jupyter Notebook
3. Run all cells sequentially

## 📌 Results

* Successfully classified tickets into categories
* Efficiently predicted priority levels
* Improved automation of support workflow

## 📌 Conclusion

This project demonstrates how NLP and machine learning can automate support ticket management, reducing manual effort and improving customer satisfaction.
